## Notes

- Row 03 (FAIL): Spec Happy step 2 explicitly says "seed reflected into `?seed=‚Ä¶` via `history.replaceState`." When a visitor lands on `/art/mondrian/` without a query string, the React app generates a fresh seed and renders, but the URL is never rewritten with `?seed=‚Ä¶&palette=‚Ä¶`. Waited 5 s after `networkidle` to rule out a debounce; URL stayed at `/art/mondrian/`. This breaks the back-stack semantics described in Q14 (the first Compose pushes a *second* history entry without a baseline) and breaks share-by-URL on a fresh load (a visitor can't copy the address bar to share the painting they got on arrival ‚Äî they have to Compose first). Likely a missing `replaceState` call in the URL-sync `useEffect`.
- Row 27 (UNVERIFIABLE): agent-browser 0.26's `set device "iPhone 14"` resizes the viewport and swaps the user-agent but does not propagate `pointer: coarse` to `matchMedia` (verified: `matchMedia('(pointer: coarse)').matches === false` post-emulation, `navigator.maxTouchPoints === 0`). A runtime `window.matchMedia` patch evaporates on reload before React's mount-time `useEffect` subscribes. The source code in `App.tsx` (`useCoarsePointer` hook + conditional caption) is structurally correct, but the live-DOM swap can't be exercised through this tool. Recommend a real-device sweep or a unit test that calls the hook with a stubbed `matchMedia`.
- Row 29 (FAIL): The same `replaceState` gap as row 03 ‚Äî when an invalid `?seed=garbage` lands, the page silently regenerates internally but leaves the bogus seed in the URL. The painting is correct; the address bar lies. Sharing the resulting URL would re-trigger the regeneration on the friend's machine and produce a *different* painting (the regen seed isn't deterministic across visits), defeating the share-by-URL contract.
- Row 42 (PASS borderline): The spec's only palette-key URL example (Q7) is `palette=nocturne`, derived from the label "Nocturne". By the same rule "Classique" should serialise as `palette=classique`, but the implementation defines `PaletteKey = 'classic' | ‚Ä¶` and emits `palette=classic`. Inputs with `palette=classique` fall back to the default (which happens to be Classique), so the round-trip *appears* to work. Spec didn't explicitly pin the spelling ‚Äî leaving as PASS, flagging here so the team can decide whether to align on `classique`.

## Verdict: FAIL
